# Gava - Java in German
Java translator from German to English

# Example code in german:

```java
öffentlich klasse Beispiel {
    öffentlich statisch leer main(String[] args) {
        zeichenkette alphabet = "abcdefghijklmnopqrstuvwxyz";
        falls (alphabet.istGleich(alphabet)) {
            System.out.drucken("wahr");
        } sonst {
            System.out.drucken("falsch");
        }
    }
}
```

# All supported German words:
`während` - `while`

`falls` - `if`

`wahr` - `true`

`falsch` - `false`

`öffentlich` - `public`

`privat` - `private`

`geschützt` - `protected`

`statisch` - `static`

`leer` - `void`

`stopp` - `break`

`weiter` - `continue`

`wechsel` - `switch`

`klasse` - `class`

`zeichenkette` - `String`

`final` - `final`

`null` - `null`

`versuch` - `try`

`fangen` - `catch`

`gleicht` - `equals`

`drucken` - `print`

`sonst` - `else`
