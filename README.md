`jeżeli` - `if`

`fałsz` - `false`

`publiczna` - `public`

`prywatna` - `private`

`chroniona` - `protected`

`statyczna` - `static`

`próżnia` - `void`

`zniszcz` - `break`

`kontynuuj` - `continue`

`przełącz` - `switch`

`klasa` - `class`

`sieć` - `String`

`finalna` - `final`

`zero` - `null`

`próba` - `try`

`złap` - `catch`

`równaSię` - `equals`

`drukuj` - `print`
